QuickMoov는 moov 아톰을 파일 앞부분으로 이동시켜 MP4, MOV, M4V, M4A, 3GP 파일을 즉시 스트리밍 가능하도록 최적화합니다.

FAST-START란?

동영상을 녹화하거나 내보낼 때, 메타데이터(moov 아톰)는 보통 파일 끝에 위치합니다. 이 경우 시청자는 재생 전에 전체 파일을 다운로드해야 합니다.

Fast-start 최적화는 이 메타데이터를 앞으로 이동시켜 다음을 가능하게 합니다:
• 웹 브라우저에서 즉시 재생
• 더 빠른 스트리밍 시작 시간
• 점진적 다운로드 지원

기능

• 드래그 앤 드롭 인터페이스 - 동영상 파일을 끌어다 놓기만 하면 됩니다
• MP4, MOV, M4V, M4A, 3GP 형식 지원
• 변환 전 상세한 파일 분석 표시
• 불필요한 free 아톰 제거로 파일 크기 감소
• 동영상 메타데이터 표시 (해상도, 길이, 코덱, 프레임률)
• 순수 Swift 구현 - 외부 의존성 없음
• 빠른 접근을 위한 메뉴 바 모드
• 보안을 위한 앱 샌드박스 활성화

사용 방법

1. QuickMoov에 동영상 파일을 드롭하세요
2. 파일의 최적화 상태를 즉시 분석해서 보여줍니다
3. 변환 버튼을 클릭하면 최적화된 사본이 생성됩니다
4. 원본 파일은 그대로 유지됩니다

콘텐츠 크리에이터, 웹 개발자, 온라인에서 동영상을 공유하는 모든 분들께 완벽한 앱입니다.
